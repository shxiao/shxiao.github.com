redis

基本

介绍

redis一个键值数据库，主要用于缓存

连接Redis服务器

ip+port, 默认端口是3679

命令分类

查询，添加，删除，设置超时，阻塞版本,对多个键值进行操作的版本

键值相关的

对键进行的操作，例如检查键是否存在exists [key], 获取所有的健keys [pattern] 删除键，获取键相关的信息

String

使用Set Get相关的命令操作

List

String列表 使用Lpush,Lpop等操作完成，List使用链表实现的，所以添加删除相关操作比较快，但是访问比较慢

Sets

无序集合 使用SADD添加新的元素到集合中

Hashes

类似于HashMap的结构
